Terra: A New Era

Indie Sandbox Voxel Game — Development Log

Terra is an experimental voxel sandbox game built in Java + OpenGL. The goal is to create a world that feels alive — natural terrain, atmosphere, and systems that evolve over time.

This site is the public dev log: updates, patch notes, performance improvements, new systems, and progress milestones as Terra grows into a full experience.

If you’re here early — welcome. You’re seeing the foundations being built.

About The Project

Terra is an experimental sandbox voxel game currently in development. This website documents the evolution of the project — including development updates, technical improvements, new systems, and progress milestones.

Development Updates

Update 0.0.8

January 27, 2026
  • Reworked the Player Movement System (v2.0), improving movement, jumping, and shifting mechanics for smoother and more responsive controls.
  • Upgraded the Procedural Tree Generator from v1.0 to v4.0, significantly improving tree variation and generation logic.
  • Improved the Water System & Water Mesh (v2.0 → v3.5), resulting in better performance and reduced rendering lag.

Update 0.0.7

January 23, 2026
  • Added hover tooltips for blocks in the inventory and hotbar (v1.0).
  • Implemented the first Biome System (v1.0), introducing Plains, Forest, and Mountain biomes.
  • Implemented the first version of the Procedural Tree Generator (v1.0).
  • Added Block Rotation support, initially implemented for the oak_log block.
  • Introduced the oak_log block with proper model and textures.
  • Expanded block variety with new blocks:
    • Oak Log
    • Oak Planks
    • Oak Leaves
    • Oak Dense Leaves

Update 0.0.6

January 15, 2026
  • Introduced a 3D Model System with support for Blockbench models and textures. Models are now rendered in-game (currently limited to 16×16×16 block scale).
  • Implemented a dynamic Sky & Day/Night Cycle System, including sun and moon rendering, smooth sunrise and sunset transitions, and a full 20-minute cycle (10 minutes day / 10 minutes night).
  • Created a Creative Inventory displaying all blocks automatically, ordered dynamically based on their JSON configuration.
  • Added a basic Hotbar System (slots 1–9) with initial textures and functionality.

Update 0.0.5

January 8, 2026
  • Introduced a JSON-based Block System (v1.0), allowing blocks to be added without modifying engine code.
  • Upgraded Camera Movement System 2.0.
  • Introduced better Water System & Water Mesh v2.0.
  • Introduced Terrain Generator System v3.0.

Update 0.0.4

January 2, 2026
  • Introduced Water Shader v1.0 with dynamic rendering.
  • Implemented a separate Water Mesh system independent from chunk meshes.
  • Reworked Chunk Mesh System v2.0 for improved performance.
  • Expanded block variety with new terrain blocks:
    • Water
    • Andesite
    • Podzol
    • Coarse Dirt
    • Moss Block

Update 0.0.3

December 30, 2025
  • Retextured all blocks for visual consistency.
  • Added block selection highlighting.
  • Implemented block breaking and placement mechanics.
  • Introduced Terrain Generator System v2.0.

Update 0.0.2

December 27, 2025
  • Fixed camera movement bug.
  • Added FPS counter and coordinate display.
  • Added new blocks: Dirt and Stone.

Update 0.0.1

December 26, 2025
  • Added basic camera movement.
  • Added infinite flat terrain generator.
  • Created the first block prototype.
