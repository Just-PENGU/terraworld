Terra World

Indie Sandbox Voxel Game — Development Log

Terra is an experimental voxel sandbox game built in Java + OpenGL. The goal is to create a world that feels alive — natural terrain, atmosphere, and systems that evolve over time.

This site is the public dev log: updates, patch notes, performance improvements, new systems, and progress milestones as Terra grows into a full experience.

If you’re here early — welcome. You’re seeing the foundations being built.

About The Project

Terra is an experimental sandbox voxel game currently in development. This website documents the evolution of the project — including development updates, technical improvements, new systems, and progress milestones.

Development Updates

Update 0.0.4

January 2, 2026
  • Introduced Water Shader v1.0 with dynamic rendering.
  • Implemented a separate Water Mesh system independent from chunk meshes.
  • Reworked Chunk Mesh System v2.0 for improved performance.
  • Expanded block variety with new terrain blocks:
    • Water
    • Andesite
    • Podzol
    • Coarse Dirt
    • Moss Block

Update 0.0.3

December 30, 2025
  • Retextured all blocks for visual consistency.
  • Added block selection highlighting.
  • Implemented block breaking and placement mechanics.
  • Introduced Terrain Generator System v2.0.

Update 0.0.2

December 27, 2025
  • Fixed camera movement bug.
  • Added FPS counter and coordinate display.
  • Added new blocks: Dirt and Stone.
  • Implemented first infinite flat terrain generator.

Update 0.0.1

December 26, 2025
  • Fixed camera rotation bug.
  • Added basic camera movement.
  • Created the first block prototype.
